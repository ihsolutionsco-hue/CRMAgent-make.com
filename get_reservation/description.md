# Get Reservation by id

Obtiene los detalles de una reserva específica mediante su ID desde la API de TrackHS PMS. Requiere autenticación y devuelve la información completa de la reserva.

## ⚠️ Error Común con Agentes de IA

Si recibes el error:
```
"Array must contain at least 1 element(s)" en "config.outputParserFormat.schema"
```

**Solución**: Configura el campo `outputParserFormat.schema` en el módulo AI Agent de Make.com con al menos un campo. Ver `SOLUCION_ERROR_AGENTE.md` para más detalles.
