# Descripción de la Herramienta

Busca unidades en TrackHS usando la API de Unit Collection (`GET /api/pms/units`). Permite filtrar por características (dormitorios, baños, amenities), disponibilidad (fechas), estado, y más. Incluye soporte para paginación (page/scroll) y ordenamiento. Requiere autenticación Basic Auth con credenciales de TrackHS.
